fn crear_usuario_oauth( &self, email: Option<&str>, nombre_visible: Option<&str>, url_avatar: Option<&str>, ) -> impl Future<Output = Result<Model, AppError>> + Send

Crea un usuario para un flujo Firebase con email (sign_in_provider = "google.com" o "password").

El email es opcional (algunos providers podrian no entregarlo) y hash_contrasena queda siempre NULL: el backend no almacena passwords desde la unificacion del handshake en /auth/firebase. El nombre y avatar vienen del Firebase profile cuando el provider los entrega.

Source

fn crear_usuario_anonimo( &self, ) -> impl Future<Output = Result<Model, AppError>> + Send

Crea un usuario anonimo: sin email, sin password, sin nombre. Solo el id UUID y activo=true.
