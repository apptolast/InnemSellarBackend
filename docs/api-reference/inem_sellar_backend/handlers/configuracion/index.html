Module configuracion

Handlers de configuracion de la aplicacion.

La tabla configuracion_aplicacion es un key-value store para ajustes globales de la app (modo mantenimiento, version minima, etc.). GET = publico, POST/PUT/DELETE = requiere autenticacion (admin).

Todos los handlers usan #[endpoint] para documentacion OpenAPI.
