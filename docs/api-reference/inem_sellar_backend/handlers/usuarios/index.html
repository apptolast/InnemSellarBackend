Handlers de perfil de usuario.

Endpoints para que el usuario vea y edite su propio perfil, y para ver perfiles publicos de otros usuarios.

§Diferencia con auth.rs

auth.rs maneja registro/login/tokens (identidad). usuarios.rs maneja el perfil (datos personales editables).

Todos los handlers usan #[endpoint] para documentacion OpenAPI.
