Convierte un String recibido del cliente en el enum TipoContenido.

§Por que String en el DTO en lugar del enum directamente

Los enums de SeaORM (DeriveActiveEnum) no implementan el trait ToSchema de Salvo OAPI. Sin ToSchema, #[endpoint] no puede generar la documentacion OpenAPI del campo. Solución: recibir como String y convertir aqui. Es el mismo patron que usa votos.rs.
