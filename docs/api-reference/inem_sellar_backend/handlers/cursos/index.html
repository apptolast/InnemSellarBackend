Module cursos 

Source
Expand description

CRUD de cursos de formacion.

GET = publico, POST/PUT/DELETE = requiere autenticacion JWT.

Los handlers usan #[endpoint] (en lugar del antiguo #[handler]) para generacion automatica de documentacion OpenAPI. La diferencia clave es que #[endpoint] analiza los tipos de los parametros en tiempo de compilacion y construye el esquema OpenAPI sin necesidad de escribirlo a mano.

Los endpoints protegidos llevan security(("bearer_auth" = [])), lo que indica a Swagger UI que muestren un candado y permitan introducir un JWT para probar el endpoint directamente desde la documentacion.
