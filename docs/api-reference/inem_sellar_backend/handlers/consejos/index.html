Module consejos 

Source
Expand description

CRUD de consejos de empleo.

GET = publico, POST/PUT/DELETE = requiere autenticacion JWT.

Los handlers usan #[endpoint] para generacion automatica de documentacion OpenAPI. Los endpoints protegidos llevan security(("bearer_auth" = [])).
