Devuelve la direccion por defecto del servidor.

§Por que una funcion separada

serde requiere que el valor por defecto de un campo sea una funcion (no un literal). #[serde(default = "default_addr")] llama a esta funcion si la variable de entorno SERVER_ADDR no esta definida. Es una limitacion del sistema de macros de serde.

§Por que .to_string()

"0.0.0.0:8080" es un &str (referencia a texto en el binario), pero necesitamos un String (texto en el heap, owned). .to_string() crea esa copia en el heap.
