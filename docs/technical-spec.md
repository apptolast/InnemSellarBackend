# InemSellar Backend — Documento Tecnico

> **PLACEHOLDER**: Sustituye este contenido por el documento tecnico generado
> en claude.ai. El script init-agent-team.sh lo leera y lo pasara al Team Lead.

## Como usar este archivo

1. Abre claude.ai y describe tu proyecto en detalle
2. Pide: "Generame un documento tecnico completo para este proyecto"
3. Copia el resultado aqui
4. Ejecuta: `./scripts/init-agent-team.sh`

## Contenido recomendado del documento tecnico

- Descripcion del proyecto y objetivos
- Requisitos funcionales (endpoints, flujos de usuario)
- Requisitos no funcionales (performance, seguridad, escalabilidad)
- Stack tecnologico elegido y justificacion
- Modelo de datos (o referencia a schema.sql)
- Endpoints API con request/response esperados
- Flujos de autenticacion
- Estrategia de deployment
- Prioridades y fases de desarrollo
